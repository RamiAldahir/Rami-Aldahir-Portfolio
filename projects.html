The project showcased procedural animation logic, motion tuning, and interactive UI behaviour within a realtime environment.

  • Unreal Engine 4
  • Custom Dampening Functions
  • Procedural Animation Logic
  • Interactive UI Systems
  • Motion Tuning & Polish
